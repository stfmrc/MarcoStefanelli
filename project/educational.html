A collection of interactive applications designed to support learning about climate change, weather forecasting, and data assimilation

This page hosts a collection of interactive applications and projects designed to support learning about climate change, weather forecasting, and data assimilation. These tools are tailored for both high school and university-level lectures, making complex scientific concepts more accessible and engaging for students.

  • WebApp - Lorenz63 and Data Assimilation Tool

    This tool is designed for educational purposes and it is useful for understanding chaotic systems and how forecasts can be improved by assimilating observations. It integrates the Lorenz63 model and applies a data assimilation technique.

    The Lorenz63 model is a simplified system that exhibits chaotic behavior, making it a suitable example for studying the impact of initial conditions on forecasts. It consists of the following three ordinary differential equations

    \[ \begin{aligned} \frac{dx}{dt} &= \sigma (y - x) \\ \frac{dy}{dt} &= x (\rho - z) - y \\ \frac{dz}{dt} &= xy - \beta z \end{aligned} \]

    The model describes the idealized motion of fluid convection, and it shows how small differences in the initial state can lead to drastically different outcomes over time. In the field of weather forecasting, this sensitivity to initial conditions is a challenge for making accurate long-term predictions. For these reasons, the Lorenz63 model is a classic example which is used to illustrate the concepts of chaos theory and the limitations of weather forecasting. Edward Lorenz coined the term "butterfly effect" during a 1972 conference in Washington, D.C., to describe the Lorenz63 model's sensitivity to initial conditions.

    The tool allows users to interact with the Lorenz63 model and observe the forecast evolution over time. Model parameters, and initial conditions, can be modified to observe the impact on the forecast. The model integration is showed on an animation where in red is the true state and in blue the forecast. Modifying the initial conditions of the forecast, the user can see how the forecast diverge from the true state when the model integration goes far from initial conditions. The model parameters can also be modified.

    The data assimilation part of the tool allows the user to assimilate observations into the model to improve the forecast. The observations are computed as perturbation of the true state, randomly adding or subtracting σobs. The user can modify the σobs and observation step to see the impact on the forecast. The data assimilation technique which has been used is a simplified version of the Kalman Filter (1DKF), where the Kalman gain is defined as \[ K = \frac{\sigma^2_{\text{model}}}{\sigma^2_{\text{model}} + \sigma^2_{\text{obs}}} \] The user can also modify σmodel to directly modify the Kalman gain. The analysis equations for the Lorenz63 components are: \[ \begin{aligned} x_a &= x_f + K (x_o - x_f)\\ y_a &= y_f + K (y_o - y_f)\\ z_a &= z_f + K (z_o - z_f) \end{aligned} \] which are used to compute the analysis and update the initial conditions of the next forecast step. The user can see how the forecast is updated with the observations and how the forecast is closer to the true state when data assimilation is used.



    Questions for students are:

    • What is the difference between the true state and the forecast state?
    • Why can't we forecast the true state in real applications?
    • Why can't we have accurate weather forecasts for more than 4-5 days?
    • How does the forecast improve when assimilating observations?
    • What is the impact of observation error on the analysis and consequently on the forecast?
    • What is the impact of the observation step on the analysis and consequently on the forecast?
    • What happens if σmodel >> σobs?
    • What happens if σobs >> σmodel?
    • What happens if σobs = σmodel?
    • Why is Data Assimilation useful in weather forecasting?
    • Could you provide other applications that could benefit from Data Assimilation?


    HOW TO USE THE TOOL:

    Download the tool and unzip it. Open the folder and click on the executable file named index.html. The tool will open in your browser.

  • Project - Dal Meteo all'Emergenza Climatica: un percorso educativo tra scienza, consapevolezza e futuro

    Il progetto “Dal Meteo all’Emergenza Climatica: un percorso educativo tra scienza, consapevolezza e futuro”, sviluppato in collaborazione con il professor Simone Napolitano, è un percorso interdisciplinare di circa 20 ore rivolto alla classe 2LD del Liceo Artistico “G. Romano” di Mantova, che integra Educazione civica, Scienze, Matematica e Informatica. L’obiettivo è sviluppare negli studenti una comprensione scientificamente fondata del cambiamento climatico, distinguendo in modo rigoroso meteo e clima, contrastando misconcezioni diffuse e potenziando competenze di analisi dei dati, valutazione critica delle fonti e cittadinanza attiva. Il percorso prevede un test diagnostico iniziale; una fase di ricerca cooperativa sul tema dell'emergenza climatica con produzione di elaborati multimediali e successiva discussione; due lezioni su caos deterministico, affidabilità delle previsioni, fake news e lettura di grafici/indicatori (IPCC, Copernicus, ECMWF, NASA). La valutazione avviene tramite rubrica multidimensionale (contenuti, competenze, soft skills), evidenziando punti di forza e criticità, in particolare sull’uso delle fonti e sulle relazioni causa–effetto. Il progetto si propone come modello replicabile nel biennio e ampliabile nel triennio con approfondimenti su teoria del caos e sistemi complessi.
