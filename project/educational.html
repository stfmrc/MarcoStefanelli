A collection of interactive applications designed to support learning about climate change, weather forecasting, and data assimilation

This page hosts a collection of interactive applications designed to support learning about climate change, weather forecasting, and data assimilation. These tools are tailored for both high school and university-level lectures, making complex scientific concepts more accessible and engaging for students. Whether you are a student or an educator, these applications provide a dynamic way to explore how the Earth's climate system works and how modern meteorology operates.
